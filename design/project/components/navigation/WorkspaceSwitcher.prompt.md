SocialSuite's signature differentiator from single-brand tools: lets the agency switch between client workspaces without logging out. Always top-right of the top bar. Pair with a dropdown listing all client workspaces (HIRA Institute, HIRA Kitchen, Al Khalil Huffaz School, Al Rehman Academy) when clicked.

```jsx
<WorkspaceSwitcher workspace={{ initials: 'HI', name: 'HIRA Institute' }} onClick={openSwitcher} />
```
